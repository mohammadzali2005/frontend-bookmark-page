# Bookmark Landing Page

A clean and functional landing page for a Bookmark Manager. This project focuses on implementing a modern, responsive design with interactive UI components.

![thumbnail](https://github.com/mohammadzali2005/frontend-bookmark-page/blob/main/Thumbnail.jpg)

## 🚀 Live Demo
[Live Demo](https://mohammadzali2005.github.io/frontend-bookmark-page/)

## ✨ Features
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **Tabbed Features Section:** Interactive tabs to switch between different product highlights.
* **FAQ Accordion:** A functional frequently asked questions section.
* **Newsletter Validation:** Form input with basic validation.
* **Custom UI Shapes:** Styled using modern CSS to match the overlapping decorative backgrounds.

## 🛠️ Built With
* HTML5 & Semantic Markup
* CSS3 (Flexbox & Grid)
* JavaScript (for Tabs and Accordion functionality)

## 📂 Project Structure
* `/assets` - Contains all icons, logos, and illustration assets.
* `index.html` - The main structure of the page.
* `style.css` - Custom styling and layout logic.
* `script.js` - Logic for the FAQ accordion and feature tabs.
* `/design`
